# Exports Directory

This directory is designated for:
- Manually exported Excel/CSV reports
- Automated backup files
- Data generated by "antigravity" scripts

## Current Export Capabilities

### Web App
- **Inventory > Export**: Downloads `Inventory_Export_YYYY-MM-DD.csv` (Client-side download)
- **Purchases**: Downloads receipt images

### Scripts
- **Inventory Updates**: Logic exists in `scripts/autoUpdateInventory.js` (currently reads from `scanner/scanned_items.csv`)
